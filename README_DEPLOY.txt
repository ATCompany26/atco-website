HOW TO DEPLOY ON NETLIFY RIGHT NOW

Easy method:
1. On Netlify's "Deploy your first project" page, use "Upload your project files".
2. Upload this ZIP only if Netlify accepts project ZIPs for build projects.
3. If Netlify treats it as a static upload and does not build it, use the GitHub method below.

Recommended GitHub method:
1. Create a free GitHub account.
2. Create a new repository called: atco-website
3. Upload every file from this unzipped folder to that repository.
4. In Netlify choose "Import from Git" → GitHub.
5. Select atco-website.
6. Netlify should automatically use:
   Build command: npm run build
   Publish directory: out
7. Deploy.
8. Add custom domain: atharvatradingco.com
9. Copy the DNS values Netlify shows into Squarespace Domains.

LOCAL PREVIEW (optional)
1. Install Node.js 20.
2. Open the folder in Terminal.
3. Run: npm install
4. Run: npm run dev
5. Open: http://localhost:3000
